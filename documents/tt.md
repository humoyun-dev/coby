# COBY ESG Corporate Landing Page -- Technical Task (TT)

Version: 2.0\
Design Direction: ESG Corporate / Institutional / Infrastructure Grade\
Status: Production Specification

------------------------------------------------------------------------

# 1. Project Objective

## Primary Goal

Position COBY as a sustainable recycling infrastructure platform, not an
eco app.

## Strategic Positioning

COBY is: - Urban environmental infrastructure - Retail-integrated
incentive system - ESG-aligned data-driven platform -
Institutional-grade technology solution

The landing page must communicate: - Credibility - Scalability -
Governance - Measurable impact - Retail and brand integration value

------------------------------------------------------------------------

# 2. Target Audience

1.  Retail Executives (Korzinka-level)
2.  ESG / CSR Managers
3.  Government Stakeholders
4.  Institutional Investors
5.  Strategic Brand Partners

Tone must be: - Serious - Data-driven - Institutional -
Technology-forward

Avoid playful eco startup styling.

------------------------------------------------------------------------

# 3. Visual Design Direction (ESG Corporate)

## Visual Tone

-   Premium
-   Minimal
-   Institutional
-   Clean geometry
-   Subtle motion

No cartoon illustrations.\
No floating playful objects.\
No overly bright gradients.

------------------------------------------------------------------------
